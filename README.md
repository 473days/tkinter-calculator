Taschenrechner Projekt

Ein einfacher Taschenrechner, der mit Python und Tkinter erstellt wurde.
Funktionen

## Grundlegende Rechenoperationen (Plus, Minus, Mal, Geteilt)
- Komma für Dezimalzahlen
- Clear-Taste zum Löschen
- Gleich-Taste für das Ergebnis

## Wie man es benutzt
1. Stellen Sie sicher, dass Python installiert ist
2. Führen Sie aus: python calculator.py

## Anforderungen
- Python 3.x
- Tkinter
