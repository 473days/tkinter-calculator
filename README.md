Taschenrechner Projekt

Das ist mein erstes Python Projekt mit Tkinter.

Ich habe einen einfachen Taschenrechner programmiert. Er kann grundlegende Rechenoperationen durchführen.

Funktionen

    Plus, Minus, Mal, Geteilt

    Zahlen von 0 bis 9

    Komma für Dezimalzahlen

    Clear Taste zum Löschen

    Gleich Taste für das Ergebnis

Wie man es benutzt

    Python muss installiert sein

    Die Datei calculator.py runterladen

    In der Kommandozeile: python calculator.py

    Der Taschenrechner öffnet sich

    Einfache Fehlerbehandlung
